## CI/CD

このプロジェクトはGitHub Actionsを使用してCI/CDを実装しています。
以下のイベントでパイプラインが実行されます：

- mainブランチへのプッシュ
- プルリクエストの作成/更新

パイプラインには以下のステップが含まれます：
1. コードのチェックアウト
2. 依存関係のキャッシュ
3. ビルドとテスト
4. リンター実行
